## 🚀 Selected Work

**[jpluggagego](https://github.com/chengyileeca-byte/jpluggagego-public)** &nbsp;·&nbsp; `TypeScript` `Next.js` `Cloudflare Workers`
My main project — a live product I designed, built, shipped, and still operate solo: Japan
luggage-delivery price comparison plus self-guided travel curation across all 47 prefectures.
JSON-as-database (no DB tax), scheduled scrapers, a feedback → GitHub Issues pipeline, and a
full migration off Vercel to Cloudflare Workers (OpenNext) to run the whole thing at near-zero
cost. Real product decisions, end to end — not a tutorial clone.

**[equity-thesis-agent](https://github.com/chengyileeca-byte/equity-thesis-agent)** &nbsp;·&nbsp; `Python` `Claude Opus 4.8`
An LLM research agent that turns a stock ticker into an 8-dimension investment thesis — with
**citations enforced as a schema constraint**, not a hope. Every claim must point to a labelled
evidence ID, so the model structurally *can't* invent a source. An eval harness scores
groundedness, honest-refusal on thin evidence, and cross-run consistency: **zero hallucinated
citations, 75% correct refusal** when the data isn't there. The part of AI I find most
interesting isn't making it talk — it's making it accountable.

**[restaurant-analytics](https://github.com/chengyileeca-byte/restaurant-analytics)** &nbsp;·&nbsp; `Python` `FastAPI` `Next.js` `Supabase`
A POS-data dashboard for restaurant owners: menu-engineering 4-quadrant analysis
(Stars / Plowhorses / Puzzles / Dogs), RFM customer segmentation, and churn detection — turning
a raw CSV export into decisions an owner can act on.

**[perfcheck](https://github.com/chengyileeca-byte/perfcheck)** &nbsp;·&nbsp; `Python` `cross-platform`
Hardware health diagnostics that measure *throughput droop under sustained load* — because aging
and dried thermal paste don't show up in a burst benchmark, they show up in degradation over
time. Cross-platform GPU monitoring, single-executable build, CI/CD.
